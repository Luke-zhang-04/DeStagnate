Errors Reference

In an effort to reduce bundle size, error messages have been moved here.

Code 1

Error code 1

0 Parameters.

Raised in: 

DeStagnate.Component

"Do not mutate state directly. Use setState instead."

You likely mutated state by doing 

this.state = {}
somewhere that is not your constructor. Use the 
this.setState({})
method instead. To disable this, invoke 
this.disableStrict()
.

Code 2

Error code 2

0 Parameters.

Raised in: 

DeStagnate.Component

"Parent is not defined. Set parent with the parent setter or set it during mounting."

You have not defined a parent element for your component. Define a parent on instantiation (in the constructor), or when you call the 

this.mount()
method (pass as param).

Code 3

Error code 3

0 Parameters.

Raised in: 

DeStagnate.Component

"Expected render method to be included in component class, no render method found, or render returned an empty array"

Your component class needs to implement the 

this.render()
method. If you're using TypeScript, you might get the message "Non-abstract class 
myComponent
does not implement inherited abstract member 
render
from class 
DeStagnate<Record<string, unknown>, Record<string, unknown>>.
"

Code 4

Error code 4

0 Parameters.

Raised in: 

DeStagnate.createElement
, and other simillar functions.

"Cannot use non-HTMLElement as component parent"

When you passed in a DeStagnate component as a child, it's parent was not an HTMLElement.
